# Copywriting: Voice & Tone Guidelines

When generating text for the HRWest website, adhere to this brand persona.

## Brand Personality Keywords
* Professional and authoritative, yet approachable
* Forward-thinking and innovative
* Inspiring and energetic (it's a premier HR conference!)
* Clear, direct, and efficient

## Grammar & Mechanics rules
* **Headings:** Use Title Case for main headings (H1, H2). Use sentence case for subheadings (H3+). Headings should be punchy and bold.
* **Punctuation:** Use exclamation points sparingly to convey genuine excitement about the event, but avoid overuse in standard interface copy. Use periods for complete sentences.
* **Clarity:** Prefer active voice over passive voice. Keep sentences concise. Drive action (e.g., "Register Now", "Become a Sponsor").

## Terminology Guide

| Do Not Use | Use Instead | Context |
| :--- | :--- | :--- |
| "Utilize" | "Use" | Keep language simple and direct. |
| "In order to..." | "To..." | Cut unnecessary words. |
| "Attendees" | "HR Professionals" or "Innovators" | Use more inspiring nouns where appropriate. |
| "Click here" | "Explore [Feature/Page]" | Use descriptive call-to-actions. |
